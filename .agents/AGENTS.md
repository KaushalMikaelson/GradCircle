# Workspace Rules

## Search & Filter Bar Design
- Keep the search & filter bar (`.program-filter-bar`, `.program-search-box`, `.program-search-input`, `.program-filter-pills`, `.filter-pill`) locked to the dark minimal aesthetic matching the reference mockup:
  - Container background: `#0d111a` with `#1c2232` subtle border and `14px` border-radius.
  - Search input: `#121722` background, `#232938` border, `8px` border-radius, pure text placeholder `"Search industries, projects, or keywords..."` without icons.
  - Category filter pills: `#94a3b8` inactive text, `#323949` active state background button with white text (`#ffffff`), `8px` border-radius.
